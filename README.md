# Desafio front-end

Aplicação que mostra livros e seus respectivos autores.

Funcionalidades:

- Livro:

  - Cadastrar um livro;
  - Visualizar os livros cadastrados;
  - Visualizar um livro específico;
  - Excluir um livro;
  - Editar um livro;

- Autor:

  - Cadastrar um autor;
  - Visualizar os autores cadastrados;
  - Visualizar um autor específico;
  - Excluir um autor;
  - Editar um autor;

Modelos:

- Livro:

  - id (obrigatório);
  - name (obrigatório);
  - author_id (obrigatório);
  - pages (opcional);

- Autor:

  - id (obrigatório);
  - name (obrigatório);
  - email (opcional);

Bibliotecas utilizadas:

- Radix UI;
- React Hook Form;
- Zod;
- Styled Components;
